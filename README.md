# school_math
analysis using school algebra and real warehouse data

Учебный проект по прикладной, школьной математике + анализ данных на основе опыта в сфере автозапчастей.

Цель:
    - Применение школьной математики к реальным задачам склада
    - Строить портфолио для Data Science / аналитики
    - Подготовиться к более сложным темам: DSP, криптография, сетевые протоколы

Данные в проекте реальны, но без содержания коммерческой тайны(имен поставщиков, клиентов и цен на товары)
